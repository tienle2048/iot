# Smart garden
## Các chức năng chính 
- frontend 
  - Login/sign up 
  - Điều khiên máy bơm 
  - Xem lịch sử hoạt động của máy bơm
  - Xem độ ẩm và ánh sáng 
- Backend 
   - check login/sign up
   - trao đổi dữ liệu giữa frontend và adafruit
- Gateway
  - nhận/ dữ liệu từ adafruit
